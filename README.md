Una trivia sobre una viaje en el espacio

## ¿CÓMO JUGAR?

Seguir las indicaciones que se dan dentro del juego.

Cada pregunta tiene alternativas diferentes y tendran consecuecnias diferentes en el juego, los puntos son la gasolina que tienes.

CUIDADO SI SE TE ACABA LA GASOLINA!

El objetivo es llegar a la tierra sano y salvo respondiendo las preguntas correctamente
